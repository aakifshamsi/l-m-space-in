# Monetag Integration Issue Note

## Current Implementation (Lines 600-619 of public.ts)
The current code uses this pattern:
```javascript
script.src = 'https://srv.monetag.com/monetag.js';
script.onload = function() {
  if (typeof Monetag !== 'undefined') {
    Monetag.init(monetagConfig);
  }
};
```

## Standard Monetag Pattern
Monetag typically uses this pattern:
```html
<script src="//disclaimer.monetag.com/?publisher=YOUR_ID&domain=l.m-space.in"></script>
```

## Issue
The current implementation may be incorrect. The `srv.monetag.com/monetag.js` script might not be the correct endpoint for loading ads.

## Fix Needed
If the debug tool shows MONETAG_API_KEY is present but ads still don't work,
the script implementation needs to be changed to use the standard disclaimer pattern.
